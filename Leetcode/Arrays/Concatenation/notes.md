##  Concatenation of Array

### 🧠 Problem
Given an integer array nums of length n, return an array ans of length 2n where:
ans[i] = nums[i]
ans[i + n] = nums[i]

### 💡 Approach
- Create a new array of size 2n
- Copy elements twice

### ⏱ Complexity
- Time: O(n)
- Space: O(n)

### 🔑 Key Learning
- Basic array traversal
- Index manipulation
